## ğŸ¯ Project Objective
The primary objective of this project is to understand how constituency electors are distributed across India and how they are represented by different political parties and coalitions. The dashboard transforms raw election data into meaningful insights that are easy to explore and interpret.

Key questions addressed:
- How many **states** are included in the dataset?
- What is the **total number of constituency electors**?
- How are electors distributed **state-wise**?
- Which **political parties** hold the highest number of electors?
- How does elector distribution vary across **coalitions**?
